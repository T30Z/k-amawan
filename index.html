About Mayan Numbers

The Mayan number system is a vigesimal (base-20) numeral system, which is distinct from the decimal (base-10) system we use today. The Mayans used a combination of dots and bars to represent numbers, where a dot represents one and a bar represents five. Numbers were written vertically, with the lowest values at the bottom and higher values above.

Each level in the vertical arrangement represents an increasing power of 20, similar to how each place in our decimal system represents an increasing power of 10. For example:

  • The first level (bottom) represents units (1 to 19).
  • The second level represents multiples of 20 (20, 40, 60, etc.).
  • The third level represents multiples of 400 (20²), and so on.

There is also a special symbol for zero, which looks like a shell. This allowed the Mayans to perform arithmetic operations and keep track of large numbers effectively.
